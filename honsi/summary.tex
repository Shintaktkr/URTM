\section*{Abstract}
 A computation model of the reversible programming language
 \texttt{R-WHILE} is assumed to have the same computing power as
 universal reversible turing machines. However, to the best of my knowledge, 
 there has been no report on the proof. In this paper, we show
 that an arbitrary reversible Turing machine can be converted to the
 \texttt{R-WHILE} program of the same meaning, and thus the
 \texttt{R-WHILE} program of reversible universal Turing machines can be
 constructed.
